markdown语法

发表于 2016-03-29   |   分类于 博客   |     |   阅读次数

Markdown是一种极简的『标记语言』，将文本转为HTML。网上有各平台下的Markdown工具可用，也有在线的，我直接使用sublime搞定。

sublime

安装在此忽略，我用的是sublime2。

安装Package Control

  1. 按Ctrl + ` 打开console
  2. 粘贴以下代码到console并回车
  3. 重启Sublime Text 2

安装Markdown Preview

  1. 按Command + Shift + P
  2. 输入pci 后回车(Package Control: Install Package)
  3. 稍等…
  4. 输入Markdown Preview回车

编辑

  1. 按Command + N 新建一个文档
  2. 按Command + Shift + P
  3. 使用Markdown语法编辑文档
  4. 语法高亮，输入ssm 后回车(Set Syntax: Markdown)

在浏览器预览Markdown文档

  1. 按Command + Shift + P
  2. 输入mp 后回车(Markdown Preview: current file in browser)
  3. 此时就可以在浏览器里看到刚才编辑的文档了

markdown语法

强调

星号与下划线都可以，单是斜体，双是粗体，符号可跨行，符号可加空格。

我做的馅饼
是全天下
最好吃的

分割线

三个或更多-_*，必须单独一行，可含空格

标题：Atx方式

无序列表

符号之后的空格不能少，-+*效果一样，但不能混合使用，因混合是嵌套列表，内容可超长。
